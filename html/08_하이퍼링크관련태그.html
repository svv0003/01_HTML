a(anchor : 연결, 고정) 태그

      - 현재 HTML 문서와 다른 HTML 문서를 연결시켜 페이지를 이동할 수 있게 만드는 태그
      
      - a 태그로 감싸진 글 / 내용 == 하이퍼 텍스트(하이터 링크)

      - 하이퍼 텍스트는 다른 문서와 구분되는 HTML만의 핵심 기능

a 태그의 속성

      - href(HyperText REFerence) : 연결할 주소를 지정하는 속성
      - target = "_blank" : 새 탭에서 열기

페이지 내부에서 점프하기

      - 한 페이지 내부에서 지정된 id 속성을 가진 태그로 이동
        -> id(식별자) 속성 : 하나의 HTML 문서 내부에서 각각의 HTML 태그를 구분하는 식별 값
                          -> 중복되는 id가 존재하면 안 됨

        class : 중복으로 이름 사용 가능
        id    : 중복으로 이름 사용 지양하는 것이 좋음
