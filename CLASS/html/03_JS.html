Script 언어란?

    - 기본 프로그램 동작을 사용자의 요구에 맞게 수행하도록 돕는 용도의 언어이다.
    - 별도의 소스 코드 컴파일(이진수 번역)을 하지 않고,
      Interpreter(인터프리터, 한 줄씩 코드를 읽어서 해석하는 기계, 기능)을
      이용해서 코드를 바로 실행한다.

    - 장점 : 컴파일에 소요되는 시간이 없다
    - 단점 : 프로그램 실행 시간이 상대적으로 느리다

JavaScript란?

    - 웹 브라우저(크롬, 엣지, 웨일, 파이어폭스 등)에서 주로 사용하는
      인터프리터 방식의 객체 지향 프로그래밍 언어이다.
      ECMA Script 표준을 따르는 대표적인 웹 기술
      (JS 버전 이름 : ES5, ES6, ES7 ...)
    
    [작성법]
    - 브라우저 개발자 도구      -> console에서 직접 작성
    - Internal 방식으로 작성  -> HTML 내부의 script 태그 작성 
    - External 방식으로 작성  -> HTML 외부의 js 파일 작성
      연결 링크 : <script src="js 파일 위치/js 파일명.js"></script>

버튼이 클릭되었을 때 internal() 함수를 호출(실행)

JS 작성법

함수 작성법

    function : 함수, 기능

    매개변수(=parameter) : 媒介變數  중매매 개입할개 변할변 셀수
                        중간에 개입하여 값을 변하게 하는 변수 이름 위치

    function 함수이름(매개변수의 값이 없을 수 있음){
          기능에 대해서 작성하는 공간!
    }
    function 함수이름(매개변수의 값이 1개 이상일 수 있음){
          기능에 대해서 작성하는 공간!
    }
    function 함수이름(){
          기능에 대해서 작성하는 공간!
    }
    function 함수이름(name1, name2, name3 ...){
          기능에 대해서 작성하는 공간!
    }
