img 태그

      - 웹 문서에서 사진이나 그림 같은 이미지를 삽입하는 용도의 태그

      [속성]
      (속성 : 시작 태그에 작성하는 내용)

      1. src : 삽입할 이미지의 경로를 작성하는 속성 (파일 위치, 인터넷 주소)
      2. width / height : 너비 / 높이를 지정하는 속성
      3. alt(대안) : 이미지 설명을 작성하는 속성
                    -> 이미지가 출력되지 않는 경우 화면에 이미지 대신 글자 출력
                       스크린 리더가 이미지의 설명을 읽어 준다

img 예시


절대 경로로 이미지 출력하기

      절대 경로 : 변하지 않는 기준점으로부터 목표까지의 경로
      (서울 종로구 삼일대로 17길 51 5층)

      - 윈도우 기준 : C드라이브, D드라이브 -> 이 기준으로 이미지의 경로나 파일 경로를 설정

상대 경로로 이미지 확인하기

      상대 경로 : 현재 파일 / 페이지 위치를 기준으로 하여 목표까지의 경로를 나타내는 것
      (학원 1층으로 내려가서 오른쪽 매장에 가서 상품을 결제해라)

고정 크기 단위(px) : 항상 고정된 크기를 지님

가변 크기 단위(%) : 화면 또는 영역에 대한 상대적인 크기 단위


고정 + 가변 크기

      고정 크기로 설정된 태그 내부에서 일정한 크기의 이미지 또는 태그를 출력하고 싶을 때 가변 크기(%)를 이용해서 지정함
